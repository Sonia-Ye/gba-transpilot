# GBA TransPilot 更新说明

## 📋 修复内容

### 1. 文本整理功能优化
- **问题**：整理文本和剔除断行功能重复，整理后断的地方有空格
- **修复**：
  - 合并为单一功能"整理文本"
  - 保留段落结构（通过\n\n识别）
  - 只去除多余空格，不添加额外空格
  - 提示用户：整理文本会保留段落，只去除多余空格

### 2. 音频/视频功能改进
- **问题**：用词不准确，"按影棚翻译"功能不明确
- **修复**：
  - 改为"音/视频转录"（Transcript）
  - 转录结果支持时间戳显示（[00:00:00]格式）
  - 分离"转录"和"翻译"两个步骤
  - 添加"显示时间戳"选项
  - 添加"翻译转录内容"按钮

### 3. AI改写/润色功能重构
- **问题**：改写功能实际在做翻译，模式说明不清晰
- **修复**：
  - 明确标注"同语言改写，不翻译"
  - 改写模式说明：
    - **意译**：保持原意，用不同表达
    - **预编辑**：优化文本以便翻译
    - **润色**：提升语言质量
  - 添加5种风格选择：
    - 标准风格
    - 正式风格
    - 随意风格
    - 学术风格
    - 商务风格
  - 改写结果保持原语言（中文→中文，英文→英文）

### 4. 改写历史记录
- **新增功能**：
  - 保存每次改写的历史记录
  - 显示原文、改写结果、模式、风格、时间戳
  - 可以点击"使用"按钮选择历史结果
  - 支持清空历史记录
  - 最多保存10条历史记录

### 5. 后端API改进
- **新增接口**：
  - `/api/save-history` - 保存历史记录
  - `/api/rewrite` 支持style参数
- **优化功能**：
  - `clean_text()` - 保留段落，去除多余空格
  - `rewrite_text()` - 同语言改写，支持多种风格

## 🚀 使用说明

### 整理文本
1. 输入或粘贴文本
2. 点击"整理文本"按钮
3. 系统会：
   - 保留段落结构
   - 去除多余空格
   - 不添加额外空格

### 音频/视频转录
1. 上传音/视频文件或粘贴链接
2. 点击"开始转录"
3. 转录结果会显示时间戳（可选）
4. 点击"翻译转录内容"进行翻译

### AI改写/润色
1. 输入要改写的文本
2. 选择改写模式（意译/预编辑/润色）
3. 选择风格（标准/正式/随意/学术/商务）
4. 点击"开始改写"
5. 查看改写结果和历史记录

## 📝 示例

### 整理文本示例
**输入：**
```
香港   是一个   国际化大都市，
拥有   繁荣的   金融市场。

和   丰富的   文化遗产。
```

**输出：**
```
香港是一个国际化大都市，
拥有繁荣的金融市场。

和丰富的文化遗产。
```

### 音频/视频转录示例
**转录结果：**
```
[00:00:00] 欢迎来到大湾区...
[00:00:05] Welcome to GBA...
[00:00:10] 香港是一个国际化大都市...
```

**翻译结果：**
```
Welcome to the Greater Bay Area...
Hong Kong is an international metropolis...
```

### AI改写示例
**原文：** 这个项目非常好，我们应该大力推广它。

**改写结果（润色-正式）：**
这个项目表现出色，值得我们全力支持并广泛推广。

**改写结果（意译-随意）：**
这项目挺不错的，咱们多推广推广。
